    SQL

    常用的SQL语句

    • SELECT * FROM table_name
    • SELECT table_name1.column_name table_name2.column_name FROM table_name1 table_name2 多表查询
    • WHERESELECT * FROM table_name WHERE xxx=xxx 条件查询
    • WHERE…INWHERE condition IN (SELECT ...) 子查询
    • ORDER BYORDER BY column_name DESC 排序
    • GROUP BYGROUP BY column_name 分组
    • LIMIT xLIMIT n 只显示最后n行
    • LIMIT x OFFSET xLIMIT n OFFSET z 显示n条,从第z行开始
    • JOINJOIN table_name alias ON column_name=alias.column_name 内连接
    • JOINRIGHT JOIN table_name alias ON column_name=alias.column_name 右连接
    • JOINLEFT JOIN table_name alias ON column_name=alias.column_name 左连接
    • DISTINCT去重
    • SUM()求和
    • MAX()求最大值
    • MIN()求最小值
    • AVG()求平均值
    • NOW()求当前时间
    • COUNT()求个数

    JDBC(Java Database Connection)

    • JDBC提供了访问世界上所有数据库的能力(标准)
    • 它是JAVA提供的一套接口，只要数据库实现了这套接口，那么你就可以通过JDBC去访问它
    • 连接JDBC需要提供一条连接字符串

    JDBC Test

    public class Main {
    public static void main(String[] args) throws SQLException {
        String jdbcUrl = "jdbc:mysql://localhost:33066/test";
        System.out.println(verifyUser(jdbcUrl, "zhangSan", "123"));
    }

    private static boolean verifyUser(String url, String username, String password) throws SQLException {
        try (Connection connection = DriverManager.getConnection(url);
             PreparedStatement preparedStatement = connection.prepareStatement("SELECT * FROM student WHERE username = ? and password = ?")) {
            preparedStatement.setString(1, username);
            preparedStatement.setString(2, password);
            ResultSet resultSet = preparedStatement.executeQuery();
            return resultSet.next();
        }
    }
}
