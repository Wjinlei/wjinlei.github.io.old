    pyenv Python多版本管理工具的使用

    次阅读 本文共 118 字，阅读约 1 分钟
    2021-11-04 01:33:25 pyenv


    介绍

    pyenv是一个python多版本管理工具 https://github.com/pyenv/pyenv pyenv-virtualenv是一个pyenv的插件，它提供了管理Python环境的功能 你可以启用一个Python环境，在这个环境中的Python版本和依赖都是独立的

    安装

    • 安装pyenv
    curl https://pyenv.run | bash
# or
curl -L https://github.com/pyenv/pyenv-installer/raw/master/bin/pyenv-installer | bash
exec "$SHELL"
    • 安装pyenv-virtualenv
    git clone https://github.com/pyenv/pyenv-virtualenv.git $(pyenv root)/plugins/pyenv-virtualenv
exec "$SHELL"
    • 配置环境变量
    export PYENV_VIRTUALENV_DISABLE_PROMPT=1
export PYENV_ROOT="$HOME/.pyenv"
export PATH="$PYENV_ROOT/bin:$PATH"
if command -v pyenv >/dev/null 2>&1; then
    eval "$(pyenv init --path)"
    eval "$(pyenv virtualenv-init -)"
fi

    简单使用

    • 列出可用的Python版本
    pyenv install --list
    • 安装指定版本
    pyenv install 3.9.5
    • 卸载指定版本
    pyenv uninstall 3.9.5
    • 查看当前已安装的Python版本
    pyenv versions
    • 查看当前激活的Python版本
    pyenv version
    • 设置全局Python版本
    # 会写入~/.pyenv/version
pyenv global 3.9.5 2.7.6 # 可以指定多个版本
    • 设置本地Python版本
    # 会写入当前目录.python-version
pyenv local 3.9.5 2.7.6 # 可以指定多个版本
    • 设置shellPython版本
    pyenv shell 3.9.5
    • 创建虚拟环境
    pyenv virtualenv 3.9.5 my-virtual-env-3.9.5 # 创建一个3.9.5版本的虚拟环境`my-virtualenv-env-3.9.5`是虚拟环境的名称
    • 列出虚拟环境
    pyenv virtualenvs
    • 激活虚拟环境
    pyenv activate my-virtualenv-env-3.9.5
    • 取消虚拟环境
    pyenv deactivate
    • 删除虚拟环境
    pyenv uninstall my-virtualenv-env-3.9.5
# or
pyenv virtualenv-delete my-virtualenv-env-3.9.5
