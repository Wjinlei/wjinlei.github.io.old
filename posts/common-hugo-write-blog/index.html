    使用hugo来写博客

    2021-11-04 03:52:01    hugo 


    前言

    关于hugo的一切,你都可以查看其 官网 来了解 ，我建议是跟着它的 quick-start 一步一步来操作

    安装

    1. hugo通过Github发布，这里我推荐你 下载 extended版本，而且最好是.tar.gz或.zip版本的
    2. 设置环境变量，将目录添加到你的Path变量中，以便命令行能够找到hugo

    创建博客

    创建博客生成器

    hugo new site xxxxx.github.io-creator

    将目录用git管理起来

    cd xxxxx.github.io-creator 
git init

    选择一个主题

    • 有很多 主题 你可以根据喜好自己选一个，推荐选择配置简单的,我用的 cactus
    git submodule add https://github.com/monkeyWzr/hugo-theme-cactus.git themes/cactus

    配置博客

    • 这一步主要是配置config.toml，下面是参考，你也可以查看主题对应的 说明
    baseURL = "https://xxxx.github.io"
languageCode = "zh-cn"
title = "博客标题"
theme = "cactus"
copyright = "版权声明"

# 主页，文章，标签，关于.
[[menu.main]]
name = "Home"
url = "/"
weight = 1

[[menu.main]]
name = "Writings"
url = "/posts"
weight = 2

[[menu.main]]
name = "Tags"
url = "/tags"
weight = 3

[[menu.main]]
name = "About"
url = "/about"
weight = 4

[markup]
  [markup.tableOfContents]
    startLevel = 2
    endLevel = 4
    ordered = false

  [markup.highlight]
    style = "github"
    lineNos = false
    noClasses = true
    tabWidth = 4

[params]
  #css = ["css/custom.css"]
  colortheme = "white" # dark, light, white, or classic
  rss = false # generate rss feed. default value is false
  showAllPostsArchive = false # default

  # Home page settings
  description = "博客描述"
  mainSection = "posts"
  showAllPostsOnHomePage = false
  postsOnHomePage = 10
  tagsOverview = true
  tocInline = true
  showProjectsList = true
  projectsUrl = "你的Github地址"

  # https://gohugo.io/functions/format/#hugo-date-and-time-templating-reference
  dateFormat = "2006-01-02" # default

  # Post page settings
  show_updated = true # default
  showReadTime = true # default

  [params.comments]
    enabled = true
    engine = "utterances" # only disqus, utterances, and cactus_comments is supported
    [params.comments.utterances]
      repo = "你的Github用户名/xxxx.github.io"
      label = "你的Github用户名"
      theme = "github-light"

  [[params.social]]
    name = "github"
    link = "你的Github地址"
  [[params.social]]
    name = "email"
    link = "你的邮箱"

    用Markdown写文章

    hugo new posts/my-first-post.md
    • 然后打开content/posts/my-first-post.md写就OK了

    预览效果

    hugo server -D

    发布博客

    hugo -D
    • 这会在其目录生成public文件夹，记得用git管理起来

    上传到Github

    • 在Github上创建一个xxxx.github.io的仓库，xxxx是你的用户名，注意全小写
    • 然后将你的public中的内容push上去吧
    • 最后你访问xxxx.github.io应该就能看到效果了，祝你好运!
